# Data Compression Algorithms (Huffman & LZ77)

## Overview
Implemented lossless data compression algorithms including Huffman Encoding and LZ77 to reduce file size while preserving data integrity.

## Features
- Huffman encoding and decoding
- LZ77 compression technique
- Efficient handling of input data

## Approach
- Built Huffman tree for optimal encoding
- Used sliding window technique for LZ77
- Evaluated compression efficiency across datasets

## Tech Stack
C++, Data Structures, Algorithms

## How to Run
1. Compile using g++
2. Run executable
